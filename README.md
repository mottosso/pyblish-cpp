### Pyblish C++

C++ implementation of Pyblish QML

<br>

### Usage

pyblish-cpp requires..

- Windows
- Visual Studio 2015
- Qt 5.8
- pyblish-base
- pyblish-qml
- Python 2 or 3

```bash
$ git clone https://github.com/mottosso/pyblish-cpp.git
$ cd pyblish-cpp
$ build & run
```
