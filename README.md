### Usage

pyblish-cpp requires QtCreator, pyblish-base and Python.

**Build**

```bash
$ cd pyblish-cpp
$ mkdir build
$ cd build
$ qmake ..
$ nmake # windows
$ make # linux
```

**Run**

```bash
$ python -u ../app.py %cd%/release/pyblish-cpp.exe
```
